# Architecture Decision Records (ADR)

Los ADR registran las decisiones de arquitectura importantes de este proyecto:
el **contexto** en que se tomaron, la **decisión**, sus **consecuencias** y las
**alternativas descartadas**. Sirven para que cualquiera (incluido mi yo futuro)
entienda *por qué* el sistema es como es, no solo *cómo* es.
